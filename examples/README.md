# Example Usage

This directory contains example files demonstrating how to use `@phcdevworks/spectre-ui-astro` in an Astro project.

## Setup

1. Install the package:

```bash
npm install @phcdevworks/spectre-ui-astro @phcdevworks/spectre-ui
```

2. Import CSS in your layout (see `ExampleLayout.astro`)

3. Use components in your pages (see `ExamplePage.astro`)

## Files

- `ExampleLayout.astro` - Shows how to import Spectre CSS
- `ExamplePage.astro` - Demonstrates component usage
